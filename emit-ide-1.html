EMIT

Java's type safety, JavaScript's syntax, WebAssembly's speed. Write code, hit ▶, runs instantly.

Types

int — 64-bit integers. No overflow. Write 42.

num — 64-bit floats. Write 3.14.

str — strings. Write "hello".

bool — true/false. void — no return.

Syntax

let x = 10 — immutable variable

mut y = 0 — mutable variable (works at top level as globals too)

fn add(a: int, b: int) int { } — function

export fn myapp() { } — exported to JS

if (x > 0) { } else { } — conditionals

while (x > 0) { } / for (let i = 0; i < 10; i++) { }

Globals

Top-level mut variables become WASM globals — they persist across function calls:

mut counter = 0

Auto-Import — Just Call It

Any function you call that isn't defined gets auto-imported. Builtins get standard implementations. Unknown functions auto-bridge to JS — if window.myFunc exists, it just works:

alert("hello") — calls window.alert

confirm("sure?") — calls window.confirm

The compiler infers parameter types from your call. Strings are read from WASM memory automatically.

Custom Imports

For full control, declare imports explicitly:

import fn fetch(url: str) from "http"

Then provide the JS implementation when instantiating the WASM.

How It Works

Emit compiles to real WebAssembly bytecode in your browser. Full pipeline: lexer → parser → type checker → WASM binary emitter. No server, no install, one HTML file.

Strings stored in WASM linear memory with length prefix. int = i64, num = f64 under the hood. Auto-promotion when mixing types.

Export

Use the Export dropdown to download your compiled program:

.wasm — raw WebAssembly binary. Use with any WASM runtime: browsers, Node.js, Deno, Cloudflare Workers, Wasmtime, Wasmer.

.html — standalone HTML page with WASM embedded. Opens in any browser, no server needed. Includes basic std implementations.

.emit — save your source code.

.js — JavaScript module with WASM embedded as base64. Import and call from any JS project.
